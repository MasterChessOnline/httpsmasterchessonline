## How to capture screenshots

1. Open the page in Chrome at desktop width (1440×900).
2. DevTools → Cmd+Shift+P → "Capture full size screenshot".
3. Crop to 1200×800 (Maps prefers 4:3 / 3:2).
4. Save as JPG, ≤2MB.

## Re-upload cadence

Add 3 fresh photos every month so the "recent updates" badge stays lit on
the listing. Rotate the gallery quarterly.
